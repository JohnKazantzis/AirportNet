AirportNet

A java application implementing a simplified airport system.

>Coded in Java.

>This application was created as an assignment of my object oriented programming course.

>In order to run the project:
-Import the project
-Add the files provided by the link (https://sourceforge.net/projects/jung/files/) in the
 project directory. These files are the libraries of the JUNG framework. The JUNG framework 
 is used in order to create a simple graph of the connections between the airports.
